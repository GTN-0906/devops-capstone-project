# devops-capstone-project

## Description
A customer account management microservice developed as the capstone project for the DevOps course.

The service exposes a REST API that allows creating, reading, updating, deleting, and listing customers on an e-commerce platform. It is built with Python and Flask, uses a database to store basic customer information (names and addresses), and is designed to be containerized with Docker and deployed on Kubernetes.

## Features
- Create a customer account
- Read a customer account
- Update a customer account
- Delete a customer account
- List all customer accounts

## Technologies
- Python / Flask
- REST API
- Docker
- Kubernetes
